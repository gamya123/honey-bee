#  Honey Bee

Honey Bee is a fun animated single-page web application built using HTML and CSS and containerized using Docker.  
The application is served through an Nginx web server running inside a Docker container.

# Project Overview

This project demonstrates how a static frontend application can be packaged into a Docker image and deployed consistently across environments. It focuses on Docker fundamentals, GitHub version control, and clean project documentation.

## Features

- Lightweight and fast
- Dockerized using Nginx
- Easy local deployment
- Beginner-friendly DevOps project

## Technologies Used

- HTML5  
- CSS3 (Animations)  
- Docker  
- Nginx  
- Git & GitHub
